# Screenshots

This folder is referenced by the root [`README.md`](../../README.md) but is
intentionally empty in version control — no placeholder or stock images were
added on purpose. Drop real screenshots of the running site here using these
exact filenames and the root README will pick them up automatically:

| File | Section |
| --- | --- |
| `hero.png` | Hero — full-bleed opening shot with the title and CTA |
| `landing.png` | Full landing page overview (can be a tall scroll capture) |
| `menu.png` | Menu — Menú Degustación course list |
| `chef.png` | Chef — Alejandro Vega portrait section |
| `gallery.png` | Gallery — bento grid |
| `footer.png` | Footer — including the credit line |

Recommended: 1600px-wide PNG or WebP, captured at the desktop breakpoint
(`lg`, ≥1024px) with the browser chrome cropped out.
